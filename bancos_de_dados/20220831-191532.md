**Projeto Conceitual - Livraria**

Projeto Conceitual -> Projeto Logico -> Projeto Fisico

* 1. A livraria deseja manter o cadastro de clientes
* 2. Cada cliente tem endereco telefone, cpf e lista de livros que
     comprou
* 3. Um cliente pode comprar muitos livros.
* 4. Existem varios exemplares de cada livro.
* 5. A livraria compra livros de editoras.
* 6. Cada editora tem um codigo, endereco, telefone e nome do gerente.
* 7. Cada livro tem nome do autor, assunto, editora e ISBN.
* 8. Cada livro eh impresso por somente uma editora.
* 9. A livraria pode nao apresentar exemplares de uma dada editora,
     mesmo tendo seu cadastro.
